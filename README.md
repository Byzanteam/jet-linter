### eslint-vue-ts

> Install

```sh
pnpm add @byzanteam/eslint-config-vue-ts -D
```

> Usage

```js
// .eslintrc.js
module.exports = {
  extends: ['@byzanteam/eslint-config-vue-ts'],
}
```
